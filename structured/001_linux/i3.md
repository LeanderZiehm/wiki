
## Modifier Key

Throughout the guide, **Mod** refers to the configured modifier key (default: **Alt**, often replaced with **Super/⌘**).

---

## Basics

| Keys        | Action            |
| ----------- | ----------------- |
| Mod + Enter | Open new terminal |
| Mod + j     | Focus left        |
| Mod + k     | Focus down        |
| Mod + l     | Focus up          |
| Mod + ;     | Focus right       |
| Mod + a     | Focus parent      |
| Mod + Space | Toggle focus mode |

---

## Moving Windows

| Keys            | Action            |
| --------------- | ----------------- |
| Mod + Shift + j | Move window left  |
| Mod + Shift + k | Move window down  |
| Mod + Shift + l | Move window up    |
| Mod + Shift + ; | Move window right |

---

## Modifying Windows

| Keys    | Action                          |
| ------- | ------------------------------- |
| Mod + f | Toggle fullscreen               |
| Mod + v | Split vertically                |
| Mod + h | Split horizontally              |
| Mod + r | Resize mode (see i3 User Guide) |

---

## Changing Container Layout

| Keys    | Action          |
| ------- | --------------- |
| Mod + e | Default layout  |
| Mod + s | Stacking layout |
| Mod + w | Tabbed layout   |

---

## Floating

| Keys                | Action               |
| ------------------- | -------------------- |
| Mod + Shift + Space | Toggle floating      |
| Mod + Left Click    | Drag floating window |

---

## Workspaces

| Keys              | Action                   |
| ----------------- | ------------------------ |
| Mod + 0–9         | Switch workspace         |
| Mod + Shift + 0–9 | Move window to workspace |

---

## Applications / Window Management

| Keys            | Action                       |
| --------------- | ---------------------------- |
| Mod + d         | Application launcher (dmenu) |
| Mod + Shift + q | Kill focused window          |

---

## Restart / Exit

| Keys            | Action             |
| --------------- | ------------------ |
| Mod + Shift + c | Reload config      |
| Mod + Shift + r | Restart i3 inplace |
| Mod + Shift + e | Exit i3            |
