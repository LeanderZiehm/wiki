
# Keyboard Shortcuts

## Built in 
Ctrl K + Ctrl C     Add Multi Line Comments
Ctrl K + Ctrl S     Open Keyboard Shortcuts
Ctrl alt C          Copy absolute path of file
Alt Z               Word wrap
ctrl-shift-I        autoformat/indent file
Cmd + Shift + F     format the file.
Alt+Up/Down         to slide lines up/down.

### Saw on internet but did't use yet
Ctrl [ or Ctrl ] Go Back / Go Forward  
alt+shift+o to remove unused imports.
F12 Jump to Definition
ctrl-shift-O to jump to symbol in window
ctrl-shift-T jump to symbol anywhere
ctrl-alt-E focus the folder view
Ctrl G      Go to line
Shift+Alt+Down to duplicate a line
ctrl + shift + k    quickly deleting the current line


## Custom
Ctrl alt enter      Run Command in Terminal
